<h1 align="center">Epitech Mouli Bot</h1>

An Epitech discord bot to notify when a new mouli arrives.

## Install

```sh
git clone git@github.com:Mazettt/HUB-epitech_mouli_bot.git epitech_mouli_bot
cd epitech_mouli_bot
```

## Usage

```sh
python src/main.py
```
